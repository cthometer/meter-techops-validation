Site Intelligence Workflow

🚀 Generate a Site Briefing Packet

1
Click "Copy Briefing Prompt" on any site card — copies a pre-filled query to clipboard.
2
Open Cowork (Claude) and paste the prompt. No additional setup needed.
3
Claude searches Airtable, Slack, Intercom, and Google Drive, then generates a full briefing packet with circuit status, port map links, tech contacts, and a readiness score.
4
The briefing is pushed to GitHub Pages and a shareable link is returned.

🔍 Query Any Site Directly

Open Cowork and use any of these prompts:
Pull site brief for [Customer / Address / Job ID] Run field validation for [Site Name] Show me all ISP blockers this week What sites are active today? Refresh the pipeline hub Check circuit status for [Job ID]
💡 Tip: Claude also pulls port map links from the Work Order (Rack Elevation & Patch Panel Layout field) and flags if circuit fulfillment is missing so you can reach out to the circuit team proactively.
